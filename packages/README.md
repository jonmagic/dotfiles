# dotfiles packages

This folder contains “single-repo” packages that are committed as part of `~/.dotfiles`.

Current shape:

- `core/`: shared host-agnostic logic
- `vscode/`: VS Code extension
- `raycast/`: Raycast extension (scaffold)

## Development

- Lint: `bun run lint`
- Typecheck: `bun run typecheck`
- Test: `bun run test`
